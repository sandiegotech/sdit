# CS 220 — Machine Learning I

**Computation & Intelligent Systems** · 4 units · Required core — everyone takes it

## The Two Registers

- **Catalog register:** Foundations of machine learning — models, training, evaluation, limits.
- **Teaching frame:** What a model is, how it trains, what it can and cannot know.

## Outcomes Served

- **Command of Intelligence**
- **Judgment**

## What This Course Is About

The machine that learned to finish your sentences is not a mystery — it is matrices, gradients, and data. Build small models from scratch so the big ones stop being magic.

## The Made Thing

A model trained from scratch on data you gathered — with an honest evaluation of where it fails.
